# DC Decision Science Operating System

## Purpose

The DC Decision Science Operating System (DCDS-OS) defines how we learn, build, document, and think.
Its purpose is to ensure that every lesson, project, case study and recommendation follows a consistent standard of quality, thinking, documentation, and execution, while supporting the mission of DC Decision Science.

## Session Rhythm

Ever apprenticeship session follows the same structure:
    1. Mission
    2. One Big Idea
    3. One Technical Concept
    4. One Deliverable
    5. Reflection
    6. Commit

## Investigation Philosphy
- Seek to understand the operational context before analyzing the data.
